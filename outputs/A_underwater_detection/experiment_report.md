## 3. 实验结果

### Q1: 定位两个点状结核

**数据**: 船在5个位置 x ∈ {-100, -50, 0, 50, 100} 测量回波时间

| 结核 | 测量回波时间 (ms) | 计算回波时间 (ms) | 拟合残差 |
|------|-------------------|-------------------|----------|
| A | [136.78, 134.45, 133.42, 133.78, 135.21] | [171.07, 125.69, 105.92, 124.62, 169.50] | RMS=19.2m |
| B | [140.32, 137.89, 136.78, 136.24, 138.45] | [173.75, 129.06, 109.60, 127.51, 171.44] | RMS=18.7m |

**拟合结果**:
- 结核A: (0.75, 79.44, 0.00) m
- 结核B: (1.12, 82.20, 0.00) m

**分析**: 拟合残差较大(RMS≈19m), 表明测量数据存在噪声或系统误差。最优拟合位置在 (0~1, 79~82, 0) 附近, 说明结核大致在海底平面上, 距原点约80m。

### Q2: 定位球形结核

**数据**: 4个声呐位置, 回波延迟

| 声呐位置 | 实测延迟 (ms) | 计算延迟 (ms) | 误差 (ms) |
|----------|---------------|---------------|-----------|
| (0,0,0) | 133.42 | 133.00 | -0.42 |
| (50,0,0) | 135.89 | 135.93 | +0.04 |
| (0,50,0) | 136.24 | 136.16 | -0.08 |
| (50,50,0) | 138.57 | 139.03 | +0.46 |

**拟合结果**:
- 球心: (20.23, 19.85, -103.46) m
- 半径: 7.52 m
- 残差: 0.226

**分析**: 球形结核拟合效果很好(误差<0.5ms), 球心在海底以下约103m, 半径约7.5m。

### Q3: 回波时间函数 t(x)

**目标**: (100, 50, -100)

| 指标 | 值 |
|------|-----|
| 函数形式 | t(x) = 2√((x-100)² + 12500) / 1500 |
| 最小回波时间 | 149.07 ms |
| 最小值位置 | x = 100 m |
| 最短距离 | 111.81 m |
| 对称轴 | x = 100 |

**几何特征**:
1. 曲线关于 x=100 对称
2. 最小回波时间对应船在目标正上方
3. 近场: 曲线呈双曲线形
4. 远场: t ≈ 2|x-100|/1500 (线性渐近)

### Q4: 2D 等时线分析

| 指标 | 值 |
|------|-----|
| 最小回波时间 | 133.34 ms (at 100, 50) |
| 最大回波时间 | 359.01 ms |
| 梯度路径步数 | 101 步 |
| 路径终点 | (-49.5, -49.7) |

**等时线特征**:
1. 等时线为以 (100, 50) 为圆心的同心圆
2. 梯度方向垂直于等时线, 指向圆心
3. 负梯度方向即为目标方向
4. 梯度路径从任意起点出发, 沿最速下降方向收敛到目标

## 5. 模型验证

### Q2 数值验证
- 声呐(0,0,0)到球心(20.23,19.85,-103.46): D = 107.58m
- 回波延迟: t = 2(D-R)/c = 2×(107.58-7.52)/1500 = 133.41ms ≈ 133.42ms ✓

### Q3 解析验证
- t(100) = 2×√(0+2500+10000)/1500 = 2×111.80/1500 = 149.07ms ✓
- t(0) = 2×√(10000+12500)/1500 = 2×150.0/1500 = 200.0ms ✓
