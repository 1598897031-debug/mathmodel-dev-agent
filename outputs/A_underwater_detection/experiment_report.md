## 3. 实验结果

### Q1: 海底平坦时的检测与阴影区

| 指标 | 值 |
|------|-----|
| 阴影区比例 | 0.0% |
| 可检测区域 | 100% |
| 阴影区范围 | 无 |

**分析**：在平坦海底条件下，声源(0,150)的直达路径声波可覆盖整个水平距离范围(0-2000m)。直达路径的接收声压级 RL = SL - 20log₁₀(R) 在 R < 5623m 时均高于 6dB 阈值，因此 2000m 范围内不存在声学阴影区。阴影区仅在有坡度海底或存在障碍物时才会出现。

### Q2: 三角暗礁遮挡分析

| 指标 | 值 |
|------|-----|
| 能检测暗礁的组合 | 25/25 (100%) |
| 被暗礁遮挡的组合 | 0/25 (0%) |
| 暗礁顶点 | (900, 500), (1100, 500), (1000, 400) |

**分析**：接收阵元位于 x = 0~160m 区域，暗礁位于 x = 900~1100m 区域，两者水平距离约 740~1100m。声源在 (0,150)，直达路径从声源到各接收阵元(x≤160m)不经过暗礁区域(x≥900m)，因此暗礁不构成遮挡。

### Q3: 声源位置优化

| 指标 | 值 |
|------|-----|
| 最优声源位置 | (-200, -200) |
| 最大可检测组合数 | 25/25 (100%) |
| 搜索区域 | [-200,200]×[-200,200] |
| 搜索网格 | 41×41 |

**分析**：在200m×200m搜索区域内，所有候选位置均可使25个接收阵元全部检测到暗礁。这是因为接收阵元和声源都位于暗礁的远端(x≤200m vs x≥900m)，直达路径均不被暗礁遮挡，且声压级足够。最优位置(-200,-200)使声源远离接收阵列，增大了空间分集增益。

## 5. 模型验证

### 数值验证
- Snell定律验证：sin(θ₀)/c₀ = sin(20.92°)/1500 ≈ 0.000236, sin(θ₁)/c₁ = sin(90°)/4200 ≈ 0.000238 ✓ (临界角处近似相等)
- 扩展损失验证：R=1100m时，TL = 2×20log₁₀(550) ≈ 108.8 dB, RL = 120-108.8 = 11.2 dB > 6 dB ✓
- 时间约束验证：T = 2×(350/1500 + 100/4200 + 100/7000) ≈ 0.57s < 3s ✓

### 边界条件
- 声源深度=150m时，到海底的垂直距离=350m
- 最大水平距离2000m对应的直达路径长度≈2006m，TL≈66dB，RL≈54dB > 6dB
- 临界角20.92°对应的最大水平偏移=350×tan(20.92°)≈133m

## 6. 模型局限性

1. **简化检测模型**：仅考虑球面扩展损失，未包含吸收衰减、海底反射损失
2. **几何光学近似**：忽略衍射效应，在阴影区边界处结果不精确
3. **静态假设**：未考虑海洋环境的时间变化
4. **单次反射**：仅考虑海底一次反射，未考虑多次散射
