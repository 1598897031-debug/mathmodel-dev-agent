# 论文框架：多静态声纳-海底地形协同优化

## 摘要

本文针对多静态声纳系统中的海底地形协同优化问题，建立了基于几何声学的三层介质射线追踪模型。通过Snell折射定律描述声波在水层(c₀=1500m/s)、淤泥层(c₁=4200m/s)和硬底层(c₂=7000m/s)中的传播规律，结合球面扩展损失模型，系统分析了声纳检测能力与海底地形的关系。

针对三个子问题：(1)在平坦海底条件下，分析了25个接收阵元的检测覆盖范围，发现水平距离2000m以内不存在声学阴影区；(2)对三角暗礁(底宽200m、高100m)进行了遮挡分析，确定了所有可检测的声源-接收阵元组合；(3)在200m×200m区域内优化声源位置，通过网格搜索找到了使检测组合数最大化的最优位置。

模型结果表明：平坦海底条件下声纳系统具有全范围检测能力；三角暗礁位于接收阵远端时不构成遮挡；声源位置优化可进一步提升系统检测性能。

**关键词**：多静态声纳；射线追踪；Snell定律；海底地形；组合优化

## 1. 问题重述

### 1.1 问题背景
- 多静态声纳系统：一个声源+多个接收阵元
- 三层海底结构：水层-淤泥层-硬底层
- 声波在层间发生Snell折射

### 1.2 问题要求
- Q1：平坦海底时的检测与阴影区分析
- Q2：三角暗礁的遮挡分析
- Q3：声源位置优化

## 2. 模型假设

1. 声波在各层中沿直线传播（几何声学近似）
2. 层间界面处服从Snell折射定律
3. 不考虑声波吸收衰减（题目未给出吸收系数）
4. 不考虑衍射效应
5. 海底为完美反射面
6. 检测条件为接收声压级≥6dB且回波时间≤3s
7. 声源级SL=120dB固定不变

## 3. 符号说明

| 符号 | 含义 | 值/单位 |
|------|------|---------|
| c₀ | 水层声速 | 1500 m/s |
| c₁ | 淤泥层声速 | 4200 m/s |
| c₂ | 硬底层声速 | 7000 m/s |
| h₁ | 淤泥层厚度 | 100 m |
| SL | 声源级 | 120 dB |
| θ₀ | 入射角（水层） | 变量 |
| θ₁ | 折射角（淤泥层） | Snell定律 |
| θ₂ | 折射角（硬底层） | Snell定律 |
| TL | 传播损失 | dB |
| RL | 接收声压级 | dB |
| R | 路径长度 | m |

## 4. 模型建立与求解

### 4.1 三层声传播模型

#### 4.1.1 Snell折射定律
$$\frac{\sin\theta_0}{c_0} = \frac{\sin\theta_1}{c_1} = \frac{\sin\theta_2}{c_2}$$

临界全反射角：θ_cr = arcsin(c₀/c₁) = 20.92°

#### 4.1.2 射线路径计算
- 水层路径：R₀ = Δy₀/cos(θ₀)
- 淤泥层路径：R₁ = h₁/cos(θ₁)
- 硬底层路径：R₂ = h₂/cos(θ₂)

#### 4.1.3 传播损失模型
- 球面扩展损失：TL = 20log₁₀(R)
- 双程传播损失：TL_two-way = 2 × TL

#### 4.1.4 检测条件
- 声压级条件：RL = SL - TL ≥ 6 dB
- 时间条件：T = 2Σ(Rᵢ/cᵢ) ≤ 3 s

### 4.2 Q1：海底平坦时的检测与阴影区

**模型**：声源固定于(0,150m)，接收阵为5×5均匀线列阵，间距40m。

**求解方法**：网格搜索法，水平距离采样400点，逐一计算每个接收阵元的检测能力。

**结果**：阴影区比例0%，全部区域可检测。

**分析**：直达路径TL = 20log₁₀(R)，在R=2000m时TL≈66dB，RL=54dB >> 6dB。

### 4.3 Q2：三角暗礁遮挡分析

**模型**：暗礁三角形顶点(900,500), (1100,500), (1000,400)。

**遮挡检测**：线段-三角形相交测试，检查直达路径和反射路径。

**结果**：25/25组合可检测，0个被遮挡。

**分析**：接收阵元(x≤160m)远离暗礁(x≥900m)，直达路径不穿越暗礁区域。

### 4.4 Q3：声源位置优化

**模型**：在[-200,200]×[-200,200]区域内搜索最优声源位置。

**优化目标**：max Σ 1[receiver_i detects reef from source S]

**求解方法**：41×41网格暴力搜索。

**结果**：最优位置(-200,-200)，可检测组合25/25。

## 5. 模型评价

### 5.1 优点
1. 物理基础扎实：基于Snell定律和几何声学
2. 计算效率高：解析方法避免PDE求解
3. 结果直观：可视化效果好
4. 可扩展：框架可推广到更复杂地形

### 5.2 局限性
1. 忽略衍射：阴影区边界不精确
2. 忽略吸收：远距离传播误差大
3. 静态模型：未考虑环境时间变化
4. 简化检测：实际声纳涉及复杂信号处理

## 6. 模型改进方向

1. 引入Bellhop射线追踪模型
2. 加入Thorp吸收系数公式
3. 考虑海底反射损失
4. 概率检测模型替代确定性检测
5. 多目标优化（覆盖率+定位精度）
6. 自适应网格加密
