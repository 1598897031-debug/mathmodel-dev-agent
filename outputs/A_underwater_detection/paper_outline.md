# 论文框架：水下目标探测与定位

## 摘要

本文针对深海铁锰结核的声呐探测与定位问题，建立了基于声波回波时间的几何定位模型。通过分析声呐回波时间与目标距离的关系，分别对点状结核定位、球形结核参数估计、回波时间函数推导和等时线路径规划四个子问题进行了系统求解。

针对Q1，利用非线性最小二乘法，由5个船位的回波时间数据定位了2个点状结核的坐标。针对Q2，建立了球面几何模型，通过4个声呐位置的回波延迟数据，拟合得到了球形结核的球心坐标和半径。针对Q3，推导了船沿X轴移动时回波时间t与船位x的解析函数关系，分析了曲线的对称性、极值点等几何特征。针对Q4，建立了二维等时线模型，绘制了三维曲面图和等高线图，分析了梯度场的方向特性及其在路径规划中的应用。

**关键词**：声呐定位；回波时间；非线性最小二乘；等时线；梯度路径规划

## 1. 问题重述

### 1.1 问题背景
- 深海铁锰结核: 富含稀有金属的海底矿产资源
- 多波束声呐系统: 发射超声波, 接收回波
- 核心挑战: 从回波数据反演结核的几何特征和埋藏状态

### 1.2 问题要求
- Q1: 由5个船位回波时间定位2个点状结核
- Q2: 由4个声呐位置回波延迟定位球形结核(中心+半径)
- Q3: 推导回波时间t与船位x的函数关系
- Q4: 2D等时线分析与梯度路径规划

## 2. 模型假设

1. 声速在海底沉积物中恒定: c = 1500 m/s
2. 点状结核可视为质点 (Q1)
3. 球形结核完全暴露在海底之上 (Q2)
4. 忽略声波多径传播和散射效应
5. 回波时间仅由直线距离决定: t = 2d/c
6. 海底为平面 (z = 0)

## 3. 符号说明

| 符号 | 含义 | 值/单位 |
|------|------|---------|
| c | 声速 | 1500 m/s |
| t | 回波时间 | ms |
| d | 声呐到目标距离 | m |
| (x_n, y_n, z_n) | 结核坐标 | m |
| (x_c, y_c, z_c) | 球心坐标 | m |
| R | 球半径 | m |
| D | 声呐到球心距离 | m |

## 4. 模型建立与求解

### 4.1 基础模型: 声呐回波距离

$$t = \frac{2d}{c} \Rightarrow d = \frac{tc}{2}$$

### 4.2 Q1: 点状结核定位

#### 距离模型
$$d_i = \sqrt{(x_{s_i} - x_n)^2 + y_n^2 + z_n^2}$$

#### 线性化
$$d_i^2 = x_{s_i}^2 - 2x_{s_i}x_n + (x_n^2 + y_n^2 + z_n^2)$$

令 $a = x_n$, $b = x_n^2 + y_n^2 + z_n^2$, 得线性方程组:
$$[2x_{s_i}, -1] \cdot [a, b]^T = x_{s_i}^2 - d_i^2$$

#### 结果
- 结核A: (0.75, 79.44, 0.00) m
- 结核B: (1.12, 82.20, 0.00) m

### 4.3 Q2: 球形结核定位

#### 球面几何
$$D_i = d_i + R$$
$$D_i^2 = (x_{s_i} - x_c)^2 + (y_{s_i} - y_c)^2 + (z_{s_i} - z_c)^2$$

#### 优化求解
代价函数:
$$\min_{x_c,y_c,z_c,R} \sum_{i=1}^{4} \left(\sqrt{(x_{s_i}-x_c)^2+(y_{s_i}-y_c)^2+(z_{s_i}-z_c)^2} - (d_i+R)\right)^2$$

#### 结果
- 球心: (20.23, 19.85, -103.46) m
- 半径: 7.52 m
- 拟合残差: 0.226

### 4.4 Q3: 回波时间函数

#### 解析推导
船位 $(x, 0, 0)$, 目标 $(x_t, y_t, z_t)$:

$$t(x) = \frac{2}{c}\sqrt{(x - x_t)^2 + y_t^2 + z_t^2}$$

#### 几何特征
- 对称轴: $x = x_t$
- 最小值: $t_{min} = 2\sqrt{y_t^2 + z_t^2}/c$
- 曲线类型: 双曲线
- 渐近线: $t \approx 2|x - x_t|/c$

### 4.5 Q4: 2D 等时线

#### 等时线方程
$$t(x, y) = \frac{2}{c}\sqrt{(x - x_t)^2 + (y - y_t)^2 + z_t^2} = t_0$$

等时线为以 $(x_t, y_t)$ 为圆心的圆:
$$(x - x_t)^2 + (y - y_t)^2 = \left(\frac{ct_0}{2}\right)^2 - z_t^2$$

#### 梯度
$$\nabla t = \frac{2}{cd}(x - x_t, y - y_t)$$

梯度方向垂直于等时线, 指向目标。

## 5. 模型评价

### 5.1 优点
1. 物理基础扎实: 基于声呐方程
2. 计算效率高: Q3/Q4解析解
3. 结果直观: 等时线图可视化
4. 可扩展: 可推广到多目标场景

### 5.2 局限性
1. 均匀声速假设: 未考虑声速剖面
2. 点目标假设: 忽略散射效应
3. 无噪声模型: 未考虑测量误差
