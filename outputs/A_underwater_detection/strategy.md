# 模型策略：水下目标探测与定位

## 问题类型识别

| 属性 | 值 |
|------|-----|
| 问题领域 | 深海锰结核声呐定位 |
| 核心数学工具 | 非线性最小二乘、几何分析、多元函数优化 |
| 计算复杂度 | 低-中等（Q1/Q2解析可解，Q3/Q4解析公式+可视化） |
| 关键约束 | 声速1500m/s、回波时间测量、球形目标几何 |

## 物理模型

### 声呐回波定位原理

主动声呐通过发射声波并接收目标反射回波来定位:
$$t = \frac{2d}{c}$$

其中 $d$ 为声呐到目标的距离, $c = 1500$ m/s 为声速。

### 坐标系

- 海平面为 XOY 平面 ($z = 0$)
- Z 轴竖直向上
- 探测船位于 $(0, 0, 0)$
- 目标位于海底或海底以下

## 四问建模方案

### Q1：定位两个点状结核

**模型选择：非线性最小二乘**

#### 核心思路
1. 结核为质点, 位于 $(x_n, y_n, z_n)$
2. 船在 $(x_s, 0, 0)$, 回波距离 $d = t \cdot c / 2$
3. 距离方程: $d_i = \sqrt{(x_{s_i} - x_n)^2 + y_n^2 + z_n^2}$
4. 5个方程, 3个未知数 → 超定方程组, 最小二乘求解

#### 求解方法
- 线性化: $d_i^2 = x_{s_i}^2 - 2x_{s_i}x_n + (x_n^2 + y_n^2 + z_n^2)$
- 令 $a = x_n$, $b = x_n^2 + y_n^2 + z_n^2$
- 线性方程组: $[2x_{s_i}, -1] \cdot [a, b]^T = x_{s_i}^2 - d_i^2$
- 求解 $x_n$ 和 $R^2 = y_n^2 + z_n^2$
- 假设 $z_n = 0$ (海底平面), 则 $y_n = \sqrt{R^2 - x_n^2}$

### Q2：定位球形结核

**模型选择：球面几何 + 非线性最小二乘**

#### 核心思路
1. 球心 $(x_c, y_c, z_c)$, 半径 $R$
2. 声呐到球面距离 $d_i$, 到球心距离 $D_i = d_i + R$
3. $D_i^2 = (x_{s_i} - x_c)^2 + (y_{s_i} - y_c)^2 + (z_{s_i} - z_c)^2$
4. 4个方程, 4个未知数 → 确定方程组

#### 求解方法
- 定义代价函数: $\sum_{i=1}^{4} (D_i^{calc} - (d_i + R))^2$
- 网格搜索 + 局部梯度下降优化
- 多起点确保全局最优

### Q3：回波时间函数 t(x)

**模型选择：解析推导**

#### 核心公式
$$t(x) = \frac{2}{c}\sqrt{(x - x_t)^2 + y_t^2 + z_t^2}$$

目标: $(x_t, y_t, z_t) = (100, 50, -100)$

$$t(x) = \frac{2}{1500}\sqrt{(x-100)^2 + 50^2 + 100^2}$$

#### 几何特征
- **对称轴**: $x = x_t = 100$ (目标x坐标)
- **最小值**: $t(100) = 2\sqrt{y_t^2 + z_t^2}/c = 2 \times 111.8/1500 \approx 149.07$ ms
- **曲线类型**: 双曲线 (水平截面为圆)
- **渐近线**: $t \approx 2|x - x_t|/c$ (远场)

### Q4：2D 等时线分析

**模型选择：二元函数 + 等值线 + 梯度场**

#### 核心公式
$$t(x, y) = \frac{2}{c}\sqrt{(x - x_t)^2 + (y - y_t)^2 + z_t^2}$$

#### 等时线
- 等时线 $t = t_0$ 为以 $(x_t, y_t)$ 为圆心的同心圆
- 圆半径: $r(t_0) = \sqrt{(ct_0/2)^2 - z_t^2}$

#### 梯度路径规划
$$\nabla t = \left(\frac{\partial t}{\partial x}, \frac{\partial t}{\partial y}\right) = \frac{2}{c \cdot d} (x - x_t, y - y_t)$$

- 负梯度方向指向目标
- 沿 $-\nabla t$ 移动可最快到达最小回波时间点
- 等时线的法线方向即为梯度方向

## 算法复杂度分析

| 问题 | 方法 | 时间复杂度 | 空间复杂度 |
|------|------|-----------|-----------|
| Q1 | 线性化+梯度下降 | O(n × k) | O(n) |
| Q2 | 网格搜索+优化 | O(G³ × k) | O(1) |
| Q3 | 解析公式 | O(1) | O(N) |
| Q4 | 网格计算+可视化 | O(N²) | O(N²) |

## 模型优缺点

### 优点
1. **物理基础扎实**：基于声呐方程和几何距离模型
2. **计算效率高**：Q3/Q4有解析公式，Q1/Q2数值优化快速
3. **结果直观**：等时线图和梯度路径易于理解
4. **可扩展**：可推广到多目标、多频段声呐系统

### 局限性
1. **简化声速**：假设均匀声速1500m/s，未考虑声速剖面变化
2. **点目标假设**：Q1假设结核为质点，忽略散射效应
3. **单频模型**：未考虑频率依赖的传播特性
4. **噪声忽略**：未考虑测量噪声和环境干扰
