# 模型策略：多静态声纳-海底地形协同优化

## 问题类型识别

| 属性 | 值 |
|------|-----|
| 问题领域 | 水下声学传播与声纳检测 |
| 核心数学工具 | 几何光学射线追踪、Snell折射定律、组合优化 |
| 计算复杂度 | 中等（Q1/Q2解析可解，Q3需网格搜索） |
| 关键约束 | 三层介质折射、三角暗礁遮挡、3s回波时限 |

## 三层声传播模型

### 物理模型

采用**几何声学射线追踪**方法，假设：
1. 声波在各层中沿直线传播
2. 层间界面处发生Snell折射
3. 忽略衍射和散射效应
4. 不考虑吸收衰减（题目未给出吸收系数）

### Snell定律应用

$$\frac{\sin\theta_0}{c_0} = \frac{\sin\theta_1}{c_1} = \frac{\sin\theta_2}{c_2}$$

其中：
- $\theta_0$ = 射线在水层中的出射角（相对垂直方向）
- $\theta_1$ = 射线在淤泥层中的折射角
- $\theta_2$ = 射线在硬底层中的折射角
- $c_0 = 1500$ m/s, $c_1 = 4200$ m/s, $c_2 = 7000$ m/s

### 折射极限角

$$\theta_{cr,01} = \arcsin\left(\frac{c_0}{c_1}\right) = \arcsin\left(\frac{1500}{4200}\right) \approx 20.92°$$

当 $\theta_0 > 20.92°$ 时，声波在水-淤泥界面发生**全反射**，不进入底层。

## 三问建模方案

### Q1：海底平坦时的检测与阴影区分析

**模型选择：解析几何 + 网格搜索**

#### 核心思路
1. 对每个接收阵元 $R_i$，计算其在水层中的水平位置 $x_{R,i}$
2. 对每个可能的声源水平位置 $x_S \in [0, 2000]$，计算射线路径
3. 判断射线是否被海底地形遮挡
4. 判断接收声压级是否满足检测条件

#### 射线路径计算
对于从声源 $S(0, 150)$ 到海底某点 $P(x_b, y_b)$ 的射线：

**水层路径长度：**
$$R_0 = \sqrt{x_b^2 + (150 - y_b)^2}$$

**出射角：**
$$\theta_0 = \arctan\left(\frac{x_b}{150 - y_b}\right)$$

**淤泥层折射角（Snell定律）：**
$$\theta_1 = \arcsin\left(\frac{c_1}{c_0}\sin\theta_0\right)$$

**淤泥层水平位移：**
$$\Delta x_1 = h_1 \cdot \tan\theta_1$$

**硬底层路径（如果到达）：**
$$R_2 = \frac{h_2}{\cos\theta_2}$$

#### 检测条件
$$SL - TL \geq 6 \text{ dB}$$
$$T = 2\left(\frac{R_0}{c_0} + \frac{R_1}{c_1} + \frac{R_2}{c_2}\right) \leq 3 \text{ s}$$

#### 阴影区判定
- 声源在 $(0, 150)$，水平距离 $x_S$
- 接收阵元 $R_i$ 在 $(x_{R,i}, 150 + (i-1) \times 40)$
- 如果从 $S$ 到 $R_i$ 的直达路径被海底地形遮挡，则 $R_i$ 对 $x_S$ 处的声源不可检测
- 阴影区 = 所有接收阵元都无法检测到的声源位置集合

### Q2：三角暗礁遮挡分析

**模型选择：几何遮挡检测**

#### 暗礁几何
- 底边中心：$x_{reef} = 1000$ m（水平距离）
- 底边范围：$x \in [900, 1100]$ m
- 底部深度：$y_{bottom} = 500 - 1000 \times \tan(0.8°) \approx 486.03$ m
- 顶部深度：$y_{top} = y_{bottom} - 100 \approx 386.03$ m
- 三角形顶点：$(1000, y_{top})$
- 底边两个端点：$(900, y_{bottom})$ 和 $(1100, y_{bottom})$

#### 遮挡检测算法
对于每对声源-接收阵元 $(S, R)$：
1. 计算射线从 $S$ 到海底的路径
2. 检查路径是否与三角形暗礁相交
3. 如果相交，则该组合无法检测

#### 三角形相交检测
使用**重心坐标法**或**边交点法**判断射线线段是否与三角形相交。

### Q3：声源位置优化

**模型选择：网格搜索 + 暴力枚举**

#### 搜索策略
在 $[-200, 200] \times [-200, 200]$ 区域内：
1. 以步长 $\Delta x = \Delta y = 10$ m 离散化（共 $41 \times 41 = 1681$ 个候选点）
2. 对每个候选声源位置，计算所有25个接收阵元的检测情况
3. 统计能发现暗礁的声源-接收阵元组合数
4. 选择组合数最多的位置

#### 优化目标
$$\max_{(x_S, y_S)} \sum_{i=1}^{25} \mathbb{1}[\text{receiver } R_i \text{ detects reef from source } S]$$

## 算法复杂度分析

| 问题 | 时间复杂度 | 空间复杂度 |
|------|-----------|-----------|
| Q1 | $O(N_x \times N_R)$ | $O(N_x \times N_R)$ |
| Q2 | $O(N_S \times N_R \times N_{check})$ | $O(N_S \times N_R)$ |
| Q3 | $O(N_{grid}^2 \times N_R \times N_{check})$ | $O(N_{grid}^2 \times N_R)$ |

其中 $N_x$ = 水平距离采样数, $N_R$ = 接收阵元数(25), $N_S$ = 声源候选数, $N_{grid}$ = 网格密度。

## 模型优缺点

### 优点
1. **物理基础扎实**：基于Snell定律和几何声学，符合水下声传播基本规律
2. **计算效率高**：解析方法避免了复杂PDE求解
3. **结果直观**：阴影区和检测区域可视化效果好
4. **可扩展**：模型框架可推广到更复杂的海底地形

### 局限性
1. **忽略衍射**：在阴影区边界处，实际声波会发生衍射
2. **忽略吸收**：远距离传播时吸收衰减不可忽略
3. **静态模型**：未考虑海洋环境的时间变化
4. **简化检测模型**：实际声纳检测涉及更复杂的信号处理

## 下一步优化方向

1. **引入Bellhop射线追踪模型**：考虑更真实的声传播效应
2. **加入吸收系数**：Thorp公式等经验吸收模型
3. **考虑海底反射损失**：不同底质的反射系数
4. **概率检测模型**：从确定性检测扩展到概率检测
5. **多目标优化**：同时优化检测覆盖率和定位精度
6. **自适应网格**：在关键区域加密采样以提高精度
